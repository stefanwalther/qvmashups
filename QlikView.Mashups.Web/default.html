Mashups with QlikView

The intention of this little collection of Html-Microsites is to demonstrate the capabilities of creating Mashups using QlikView documents/objects.

Motivation

It is exremely easy to create Mashups with QlikView. Unfortunately this integration capability is not really well known and also not documented in detail.
That's why I have created this microsite wich should help as

  • A reference
  • A collection of code-samples
  • A collection of ready-to-demo examples

Three Different Ways to Realize Mashups

From the technical perspective there are three completely different approaches how one could implement Mashups with QlikViews. This Mashup-microsoft tries to cover all of these three approaches. Finally I have added a comparison of these approaches which could be a starting point if you are not sure which approach meets your requirements best.

  1. Simple document integration (using opendoc.htm)
  2. Simple object integration (using singleobject.htm)
  3. Div integration (using the Ajax Javascript library)

Hints for Setting up the Examples:

Unfortunately there is - in most cases - little work to get the examples up and running.

  • For running these examples you have to ensure that you fullfill the following requirements:
    • A local installation of QlikView Accesspoint + QlikView Server
    • The local QlikView Accesspoint has to be accessible via http://localhost
    • Ensure that the following link works: http://localhost/QvAJAXZfc/opendoc.htm?document=Sales%20Compass.qvw
    • If not, download the "Sales Compass.qvw" application from demo.qlikview.com and make it accessible in the root directory of your Accesspoint
  • There is no need to open these example Html files using a web server. You can just open it from your file system. Doing so you will probably get a warning by your browser to allow execution of ActiveX ... please confirm that.

Compatibility:

  • All examples have been tested using QlikView 10 and QlikView 11.
