Simple Mashup Integration: Single Object

This example loads one object (list box) from a single QlikView document.

Javascript code placed in the header of the Html file to initialize the QlikView-object(s). 

            <script language="javascript" type="text/javascript">

                    Qv.InitWorkBench({

                           View: 'Sales Compass',

                           Host: null


                    });

            </script>

            
        .QvInlineObject
 
        {

               position:relative;

               margin-left:10px;

               margin-right:10px;

               float:left;

        }

DIV blocks to place the QlikView objects within your webpage. 

            <!-- Listbox with the Id LB5732 -->

            <div class="QvInlineObject" avqview="Sales Compass" avq="object:.Document\LB5732"></div>
