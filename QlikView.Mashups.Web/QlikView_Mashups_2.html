Introduction

Integrating single QlikView objects by using an IFRAME and singleobject.htm works very similar as the opendoc.htm integration.

The main difference between these two approaches is that
  • opendoc.htm refrences a complete QlikView application (including all sheets, toolbar, and so on).
  • singleobject.htm references a single object


On the other hand singleobject.htm integration and DIV integration is very similar:
  • Both approaches brings a single object into your existing web application
  • For a full comparison of these two approaches please visit this site

Parameters

Use the following parameters to define the behaviour how a QlikView object should be integrated into your website:

QueryString Parameter Explanation Example
document Specify the path to the document on server document=Sales%20Compass.qvw
object Specify the Id of the QlikView object object=LB1022
bookmark Specify the bookmark which should be called (using the ID of the bookmark) bookmark=Document\BM01
select Specify any selection using field names and values
(Have a look at the examples below) 		select=LB01,Germany
ticket 40 hex string that was returned by the Ticket webservice
host Host IP of the QlikView Server (optional if running on localhost) host=local

Parameter Examples

Action Example
Single selection (Germany) in listbox with ID = LB01 select=LB01,Germany
Multiple selection in one list box select=LB01,Germany,USA
Multiple selection in multiple list boxes select=LB01,Germany,Argentina&select=LB02,AnotherDimension
Specify if it's a server or document object (document is default) select=Document\LB01,Germany,USA
Select a bookmark bookmark=Document\BM01





Examples

The examples below make use of the demo application IT-Asset-Management which can be downloaded and viewed at http://demo.qlikview.com

Simple integration of an object

Calls an object from the document "IT Asset Management" with the Id "CH32" 

                    <iframe src="http://localhost/QvAJAXZfc/singleobject.htm?document=IT%20Asset%20Management.qvw&object=CH32" />

Calling a bookmark

The two examples below load the same chart ("On Order and In Stock") from the IT-Asset-Management application but with different bookmarks. 

                    <iframe src="http://localhost/QvAJAXZfc/singleobject.htm?document=IT%20Asset%20Management.qvw&object=CH378&bookmark=Document\BM01" />

Calling a bookmark

The two examples below load the same straight table ("Employee Movements") from the IT-Asset-Management application but with different selections in the field "Region" (available in the list box LB47). 

                    <iframe src="http://localhost/QvAJAXZfc/singleobject.htm?document=IT%20Asset%20Management.qvw&object=CH23&select=LB47,Americas" />
                        


                    <iframe src="http://localhost/QvAJAXZfc/singleobject.htm?document=IT%20Asset%20Management.qvw&object=CH23&select=LB47,Germany" />
                        


                    <iframe src="http://localhost/QvAJAXZfc/singleobject.htm?document=IT%20Asset%20Management.qvw&object=CH23&select=LB47,Americas,Germany" />
