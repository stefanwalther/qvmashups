Using InitialSearch

This example loads two objects from a single QlikView document.

Note that interactivity between these two objects is still supported!
(Try to select some "Forecast Categories" and the "Sales Pipeline" chart will react.)

Javascript code placed in the header of the Html file to initialize the QlikView-object(s). 

            <script language="javascript" type="text/javascript">

                    Qv.InitWorkBench({

                           View: 'Sales Compass'

                    });

            </script>

DIV blocks to place the QlikView objects within your webpage. 

            <!-- Listbox with the Id LB5732 -->

            <div class="QvInlineObject" avqview="Sales Compass" avq="object:.Document\LB5732" style="height:100px;width:150px;"></div>

            

            <!-- Chart with the Id CH351 -->

            <div class="QvInlineObject"  avqview="Sales Compass" avq="object:.Document\CH351" style="width:300px; height:250px;"></div>
