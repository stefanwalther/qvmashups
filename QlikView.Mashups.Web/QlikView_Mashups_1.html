QlikView Mashups - Document Integration (using opendoc.htm)

  • Introduction
  • Parameters
  • Example

Introduction

This example uses "opendoc.htm" which is located in the QvAJAXZfc folder of the QlikView webserver.

Parameters

Use the following parameters to define the behaviour how your QlikView application should be integrated into your website:

QueryString Parameter Explanation Example
document Specify the path to the document on server document=Sales%20Compass.qvw
bookmark Specify the bookmark which should be called (using the ID of the bookmark) bookmark=Document\BM01
select Specify any selection using field names and values
(Have a look at the examples below) 		select=LB01,Germany
ticket 40 hex string that was returned by the Ticket webservice
host Host IP of the QlikView Server (optional if running on localhost) host=local

Parameter Examples

Action Example
Single selection (Germany) in listbox with ID = LB01 select=LB01,Germany
Multiple selection in one list box select=LB01,Germany,USA
Multiple selection in multiple list boxes select=LB01,Germany,Argentina&select=LB02,AnotherDimension
Specify if it's a server or document object (document is default) select=Document\LB01,Germany,USA
Select a bookmark bookmark=Document\BM01

Code Examples:

Integrating QlikView application "Sales Compass.qvw": 

                    <iframe src="http://localhost/QvAJAXZfc/opendoc.htm?document=Sales%20Compass.qvw" />


Integrate QlikView application "Sales Compass.qvw" with bookmark "BM01": 

                    <iframe src="http://localhost/QvAJAXZfc/opendoc.htm?document=Sales%20Compass.qvw&bookmark=Document\BM01" />


Integrate QlikView application "Sales Compass.qvw", selecting the value "2009 in the listbox with the ID "LB5698": 

                <iframe src="http://localhost/QvAJAXZfc/opendoc.htm?document=Sales%20Compass.qvw&select=Document\LB5698,2009" />


Integrating QlikView application "Sales Compass.qvw", selecting the values "2009" and "2010" in the listbox with the ID "LB5698" and value "Q1" in the listbox with the ID "LB5699": 

                <iframe src="http://localhost/QvAJAXZfc/opendoc.htm?document=Sales%20Compass.qvw&select=LB5698,2008,2009&select=LB5699,Q1" />

Integrate QlikView application "Sales Compass.qvw" with bookmark "BM02" (which will in fact activate the sheet "Planning").
Note: This example only works if you create an empty bookmark (checking "Layout State") which will point the the sheet "Planning". 

                    <iframe src="http://localhost/QvAJAXZfc/opendoc.htm?document=Sales%20Compass.qvw&bookmark=Document\BM02" />
